# Fybo

Fybo es una aplicación de moda creada por los estudiantes de 2º de DAW José Carlos, Pau y Alexey.

## Instalación

Clona el proyecto.

```bash
git clone https://github.com/alexeyrudd/fypo.git
```

Instala paquetes.

```bash
npm install
```

## Uso


Con la App Expo utiliza el código QR generado.
